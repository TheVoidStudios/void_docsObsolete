Add a Blaster Rigid Body in Motive (Passive Tracking)

This process requires two people; one working with the Motive software and the other to go hold the blaster in shooting position on the stage.
  1. Have a person hold the blaster in shooting position on the stage, keeping the blaster level and aiming down the Z-axis.
  2. Log in to the Capture Server and launch the Motive software.
  3. Locate the markers for the blaster.
  4. Select the markers as a group.
  5. Right-click, select Rigid Body and then select Create From Selected Markers.

    The blaster now displays as a rigid body and it is added as an Asset in the Project menu with a default name.

  6. Save this blaster according to the naming convention.
    • GUN2##; where # is the last two numbers of the blaster's ID number (shown in the LCD or under the pump).
  7. Set the pivot point.
    1. Physically add a temporary marker on the blaster so that it is on the top of the blaster close to the LCD.

      For example:

    2. Add the pivot point in Motive.
      1. Press and hold Ctrl and then select the temporary maker.
      2. Right-click, select Rigid Body and then select Set Pivot Point to Selected Marker.
    3. Physically remove the temporary marker from the blaster.
  8. Specify the rigid body properties.
    1. Locate the information on this rigid body in the Project tab.
    2. Select Properties, and specify the following:
      • Deflection: 10
      • Tracking Algorithm: Marker-based
      • Enhance Performance: True
      • Share Markers: False
      • Min Marker Count: 4
      • User ID: the numerical number assigned to the blaster's name (for example, 201)
  9. Save this blaster rigid body.
  10. Repeat this procedure for each blaster used on this stage.
