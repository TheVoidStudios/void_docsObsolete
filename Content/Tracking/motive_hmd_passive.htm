Add a HMD Rigid Body in Motive (Passive Tracking)

  1. Place the HMD on the stage so the HMD is level, right side up and facing down the Z-axis.
  2. Log in to the Capture Server and launch the Motive software.
  3. Locate the markers for the HMD.
  4. Select the markers as a group.
  5. Right-click, select Rigid Body and then select Create From Selected Markers.

    The HMD now displays a rigid body and it is added as an Asset in the Project menu with a default name.

  6. Save this HMD rigid body according to the naming convention.
    • HMD1##; where # is the last two numbers of the HMD's serial number.
  7. Set the pivot point.
    The pivot point depicts the Z-axis (right between the eyes) of the HMD rigid body.
    1. Physically add a temporary marker at each end of the visor.

      • Second marker (left-side)

    2. Select the first temporary marker (on the right-side of the visor) and then right-click and set it as the pivot point.
    3. While still selecting the first marker, click the second marker, you will see a white line showing the distance between the two markers.
    4. Rotate the rigid body's Z-axis and Y-axis to align the white line with the X-axis (red line).
      1. In the Project menu, locate the menu for this rigid body and select the Transformation tab.
      2. In the Rigid Body Orientation area, enter an adjustment value (usually less than a degree) for the Z-axis and the Y-axis.
      3. Click Apply Rotation.
      4. In the Perspective View, determine whether further adjustment is needed.
        • If the white line is not aligned with the red line, repeat this procedure to rotate the rigid body.
        • If the white line is aligned with the red line, continue to the next step.
    5. Determine the pivot point.
      1. In the Perspective View, look at the distance measured by the white line and divide that distance in half.
      2. In the Transformation tab, in the Rigid Body Pivot-Point Translation area, set the X-axis value to the half-way distance.
      3. Click Apply Translation.
    6. Physically remove the temporary markers from the HMD.
  8. Specify the rigid body properties.
    1. Locate the information on this rigid body in the Project tab.
    2. Select Properties, and specify the following:
      • Deflection: 10
      • Tracking Algorithm: Marker-based
      • Enhance Performance: True
      • Share Markers: False
      • Min Marker Count: 4
      • User ID: the numerical number assigned to the HMD's name (for example, 101)
  9. Save this HMD rigid body.
  10. Repeat this procedure for each HMD used on this stage.
