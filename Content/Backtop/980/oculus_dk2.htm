Install the Oculus software (DK2)

  1. If you have not already done so,
    1. Connect the to your workstation.
    2. Open the BACKTOP DRIVERS AND SOFTWARE folder.
  1. Locate and execute the oculus_runtime_sdk_0.8.0.0_win.exe file.
  2. If a window displays asking if you want this program to make changes, click Yes. The wizard launches.
  3. On the first page, click Next.
  4. When prompted to accept the terms and conditions, select the radio button to accept the agreement and click Next.
  5. On the next page, click Next.
  6. On the next page, click Next. The installation process begins.
  7. If a message displays asking if you install the device, click Yes.
  8. When the installation has finished, deselect the checkbox to view the Readme file and click Finish.
