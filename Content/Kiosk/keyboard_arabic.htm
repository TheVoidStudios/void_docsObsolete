Add the Arabic Language Keyboard

  1. If you have not already done so, access the 'Settings' Window.
  2. In the 'System' area, tap Language and input then locate the 'Keyboards and inputs' area.
  3. Tap Hacker's Keyboard.

    The window for the keyboard's settings displays.

  4. Select the option for the 'Input languages'.

    The Input languages window displays.

  5. Confirm the checkbox to enable the English language is selected.
  6. Select the checkbox to enable the Arabic language.

    For example:

    Click to enlarge

  7. Save the settings.
  8. Return the kiosk to the scan screen.

    For example:

    Click to enlarge

  9. Test the keyboard.
    1. If you have not already done so, obtain a test barcode.
    2. Scan the test barcode at the kiosk.
    3. Tap NEXT to accept the default (English) language.
    4. When the keyboard displays, do the following:
      1. Confirm the keyboard color is black.
      2. Swipe across the space bar (for example, right to left) to switch to another, enabled language. The language name displays on the space bar.
      3. Confirm the keyboard for Arabic displays.

        For example:

        Click to enlarge

    5. Repeat as needed for any additional keyboard language.
  10. Repeat for every kiosk used at this site.
