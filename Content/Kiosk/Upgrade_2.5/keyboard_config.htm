Configure the Kiosk for the Hacker's Keyboard

  1. On the desktop, launch the Hacker's Keyboard application.
  2. When the application's window displays, click ENABLE KEYBOARD.

    For example:

    Click to enlarge

    The 'Settings' window displays.

  3. Enable (turn on) the Hacker's Keyboard.
  4. When a warning appears, click OK.

    For example:

    Click to enlarge

  5. Exit out of and then return to the 'Settings' window.
  6. In the 'System' area, tap Language and input.
  7. Set Hacker's keyboard to be the default keyboard.
    1. Tap Default keyboard.
    2. When the window displays, select the radio button for the Hacker's keyboard..

      For example:

      Click to enlarge

  8. Customize the keyboard.
    1. Tap Hacker's keyboard.

      The window for the keyboard's settings displays

    2. Set the keyboard's landscape height to 31%.

      For example:

      Click to enlarge

    3. Select the checkbox to 'Show soft keyboard'.

      For example:

      Click to enlarge

    4. Accept all other default settings and exit the window.
  9. Repeat for every kiosk used at this site.

If the site requires the keyboard to display multiple languages, continue with the procedure to Add a Keyboard Language.
