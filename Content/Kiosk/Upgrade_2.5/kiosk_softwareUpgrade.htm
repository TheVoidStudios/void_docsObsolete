Upgrade the Kiosk (WizardApp) Software

This procedure is performed from your workstation; not at the kiosk.

  1. If you have not already done so, open the AirWatch Console.
  2. Select APPS & BOOKS. The available versions display in list view.

    For example:

    Click to enlarge

  3. Select the WizardApp version currently assigned to the kiosks.

    The information for that version appears in 'Details View' with the 'Summary' tab selected.

  4. Select the 'Devices' tab to confirm the site's kiosks are listed.
  5. Click Add Version.

    For example:

    Click to enlarge

  6. When the 'Add Version' window displays, click Upload.
  7. When the 'Add' menu displays, do the following:
    1. Select the 'Local File' radio button.
    2. Click Choose File.
    3. Navigate and select the software version to which you want the kiosks upgraded.
    4. When the file is listed, click Save.

    The 'Add' window closes and you are returned to the 'Add Version' window.

  8. Confirm the file you added appears in the 'Application File' field.
  9. Click Save.
  10. Exit the AirWatch Console.
