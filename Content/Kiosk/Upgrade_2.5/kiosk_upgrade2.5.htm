Upgrade Kiosks to Software Version 2.5

Version 2.5 introduced the Hacker's Keyboard and removed the Socket Scan application. As a result, additional upgrade steps are required.

Do the following:

  1. Uninstall SocketScan Keyboard Wedge
  2. Unpair the Scanner
  3. Upgrade the Kiosk (WizardApp) Software
  4. Configure the Kiosk for the Hacker's Keyboard
  5. Reset the Scanner
  6. Add a Keyboard Language (if required)
