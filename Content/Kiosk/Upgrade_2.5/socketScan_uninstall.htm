Uninstall SocketScan Keyboard Wedge

The kiosk software version 2.5 also includes integration with the Hacker's Keyboard. As a result, the kiosk no longer uses the Socket Scan Keyboard Wedge application.

  1. At the kiosk, from the desktop, open the App Store.
  2. Search for the SocketScan Keyboard Wedge application select it from the search results.
  3. When the application screen displays, click UNINSTALL.

    For example:

    Click to enlarge

  4. When a message appears asking if you want to install this application, click OK.

    5. The application screen button now displays 'INSTALL'.

  5. Exit the App Store.
